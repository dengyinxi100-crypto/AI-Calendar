## 三、创建日志 / 任务

### 方式一：快捷键
按 **Alt+N** 或从侧边栏点击「+ 新建日志」打开编辑器。

### 方式二：日历点击
1. 点击日历中的任意日期
2. 在弹出的日志列表中点击「+ 创建日志」

### 方式三：AI 对话创建（推荐）
1. 点击工具栏「🤖 AI」按钮打开 AI 面板
2. 在「对话」标签中直接描述，如：**「明天下午3点开会讨论项目，每周三重复」**
3. AI 自动解析并创建周期任务（无需手动填表）

### 表单字段说明

| 字段 | 说明 |
|------|------|
| **标题** | 必填，简短描述 |
| **分类** | 会议 / 任务 / 杂务 / 个人 / 工作 / 学习 / 健康 / 其他（支持 AI 一键分类） |
| **设为任务** | 勾选后启用优先级 + 状态管理 |
| **优先级** | 紧急(红) / 次要(黄) / 非紧急(灰) |
| **日期** | 默认为当天 |
| **时间** | 开始/结束时间 |
| **内容** | 详细描述 |
| **重复周期** | 6 种模式：每天 / 工作日 / 休息日 / 每周（可选星期几）/ 每月（可选几号） |
| **提醒** | 勾选后设置提醒时间 + 提前分钟数（准时/5/15/30/60分钟/1天前） |

## 四、周期任务

### 创建周期任务
在编辑器中设置「重复周期」：

| 模式 | 效果 |
|------|------|
| **每天** | 天天出现 |
| **工作日** | 仅周一至周五 |
| **休息日** | 仅周六、周日 |
| **每周** | 指定星期几，如「每周一、周三」 |
| **每月** | 指定几号，如「每月 15 号」 |

### 自动日实例
- 每次启动应用时，系统自动为当天生成周期实例
- 每小时也会检查一次（node-cron 定时任务）
- 实例 ID 包含日期，**不会重复创建**

### 研究类任务自动处理
当周期任务标题/内容包含以下关键词时，AI 会自动完成：

> 查询、分析、总结、收集、搜索、查找、了解、研究、比较、调查、整理、统计、汇总、报告、规划、盘点、梳理、归纳、评估、解读、翻译

**示例**：「每日查询 AI 行业最新动态」→ 每天早上自动生成 + AI 分析 → 自动标记完成并填入 AI 摘要。

> 注意：会议、拜访、提交等行动类任务**不会**被 AI 自动处理，需要人工完成。

### 手动完成周期任务
在日历中点击周期任务实例 → 点击 ✓ 标记完成 → 任务日期自动推进到下一次出现。

## 五、任务管理

### 状态流转

```
待处理 (pending) → 进行中 (in_progress) → 已完成 (completed)
```

在任务面板、看板视图或日志详情中点击状态标签即可切换。

### 看板视图
按 **Alt+2** 切换到看板/时间轴视图，任务自动按时间分为 5 列：

| 列 | 说明 |
|----|------|
| 逾期 | 超过今天日期的未完成任务 |
| 今天 | 当天截止的任务 |
| 明天 | 明天截止的任务 |
| 本周 | 本周内截止的任务 |
| 稍后 | 本周之后的任务 |

点击列标题可折叠/展开，逾期列默认展开。任务卡片带呼吸灯动画（红色=逾期/待处理，黄色=进行中，绿色=已完成）。

### 任务面板
点击「📋 任务」按钮打开右侧面板：
- 查看全部任务（带呼吸灯状态指示）
- 批量勾选 + 删除（含确认弹窗）
- 快速状态切换

### 任务概览
工具栏下方展示 4 张统计卡片：全部任务 / 待处理 / 已完成 / 已逾期 + 完成率环状进度条。

## 六、AI 功能

### 支持的 AI 服务商

| 服务商 | 默认模型 | API Key 获取 |
|--------|----------|-------------|
| **DeepSeek**（推荐） | deepseek-chat | [platform.deepseek.com](https://platform.deepseek.com/) |
| **OpenAI** | gpt-4o | [platform.openai.com](https://platform.openai.com/) |
| **Anthropic Claude** | claude-sonnet-4 | [console.anthropic.com](https://console.anthropic.com/) |
| **智谱 GLM** | glm-4-plus / glm-4-flash（**免费**） | [open.bigmodel.cn](https://open.bigmodel.cn/) |
| **通义千问 Qwen** | qwen-plus | [dashscope.aliyun.com](https://dashscope.aliyun.com/) |
| **Ollama（本地）** | 自定义 | 无需 Key，本地运行 |

### 首次配置
1. 点击工具栏「🤖 AI」按钮
2. 切换到「设置」标签
3. 选择模型供应商（如 DeepSeek）
4. 输入该供应商的 API Key
5. 选择模型
6. 勾选「启用 AI 辅助功能」
7. 点击「保存设置」
8. 切换供应商时自动保存，每个供应商的 Key **独立存储互不干扰**

### AI 对话
- 支持**多轮对话**，上下文保持
- 点击「引入日志」按钮将当月日志作为对话上下文
- **全局共享对话历史**：切换供应商后对话无缝衔接，不丢失上下文
- 工具栏显示当前供应商标签（如"DeepSeek"），一目了然
- 蓝色「保存到当天日志」按钮将 AI 回复存为当天日志
- 发送过程中可随时点击「停止」中断
- 点击「清空对话」清除全部历史

### 自然语言创建任务
在对话中直接描述，AI 自动解析为结构化任务并存入数据库：

> 「下周三下午2点到4点产品评审，每周三重复，设为紧急任务」
> → 自动创建周期任务：每周三 14:00-16:00「产品评审」

### AI 总结
切换到「总结」标签，选择类型 + 日期 → 点击「生成总结」：
- **每日总结**：当天日志汇总
- **每周总结**：本周回顾
- **每月总结**：月度报告

生成后可一键「保存到当天日志」。

### AI 工作流
切换到「工作流」标签，可执行多步骤智能任务：
- 自然语言描述 → AI 解析 → 自动创建任务
- 支持多轮对话追问
- 可选择独立模型（与对话设置不同）

### 联网搜索
AI 对话内置 DuckDuckGo 搜索工具，可查询实时信息。

| 选项 | 说明 |

## 八、提醒功能

1. 编辑日志时勾选「🔔 开启提醒」
2. 设置提醒时间 + 提前分钟数（准时/5/15/30/60分钟/1天前）
3. 到达时间后弹出：
   - **桌面通知**（Windows 右下角）
   - **站内 Toast**（应用内弹窗）
4. 每 30 秒自动检查待提醒项
5. 当天已触发的提醒不会重复弹窗

## 九、数据存储

| 项目 | 位置 |
|------|------|
| **日历数据** | `%APPDATA%\AI 智能日历\calendar.db` (SQLite) |
| **主题/AI 配置** | 浏览器 localStorage |
| **AI 对话历史** | localStorage（全局共享） |
| **API Key** | SQLite 数据库（各供应商独立存储，前端不可复制） |
| **EXE 文件** | 可放在任意位置，便携运行 |

> 备份数据：复制 `%APPDATA%\AI 智能日历\calendar.db` 文件即可。恢复时放回原位置。

## 十、常见问题

### Q: AI 功能无法使用？
1. 确认已在「设置」标签中选择正确的供应商并输入 API Key
2. 确认「启用 AI 辅助功能」已勾选
3. 点击「保存设置」
4. 如显示具体错误：401→Key 错误，404→模型名错误，429→频率限制

### Q: 切换 AI 供应商后对话丢失？
不会。v1.0 起对话历史全局共享，切换供应商后对话无缝衔接。

### Q: 关闭后数据丢失？
不会。数据存储在 `%APPDATA%\AI 智能日历\` 下的 SQLite 文件中，应用关闭后持久保留。

### Q: 如何备份数据？
复制 `%APPDATA%\AI 智能日历\calendar.db` 文件即可。恢复时放回原位置。

### Q: 周期任务没有自动出现？
- 确认周期模式设置正确
- 重启应用会立即触发检查
- 每小时自动检查一次

### Q: GLM-4-Flash 真的免费吗？
是的。智谱 AI 的 GLM-4-Flash 模型永久免费，在 [open.bigmodel.cn](https://open.bigmodel.cn/) 注册即可获取 API Key。

### Q: 如何使用本地模型？
选择「Ollama（本地）」供应商，确保本地已安装并启动 [Ollama](https://ollama.com/)，然后在设置中填入模型名称（如 `qwen2.5`、`llama3.2` 等）即可，无需 API Key。
