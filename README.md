## 一、安全说明

1. **本地优先离线架构**  
   所有日程、任务、AI 密钥持久化存储于本机 `%APPDATA%` 内置 SQLite 数据库，无后台静默上传用户数据逻辑，断开网络可完整使用全部基础日历功能。

2. **AI 与数据库权限隔离**  
   大模型仅输出标准化 JSON 变更描述，无直连、直写数据库权限；后端统一执行 Schema 校验、敏感字段拦截、高危操作阻断，规避 AI 不可控输出篡改本地数据。

3. **网络请求控制**  
   AI 网络请求仅在用户主动触发对话、总结、联网检索时发起；支持 Ollama 本地离线大模型，可实现全内网无公网运行，适配涉密场景。

4. **安全基线**  
   内置 Electron 标准沙箱安全配置、全量参数化 SQL 查询防注入，但软件仅面向个人学习使用，不承诺达到金融、军工等极高等级等保标准，涉密企业商用前需自行完成安全复测与改造。

## 三、主要功能

- **日历视图**：月视图 / 周视图 / 日视图，支持拖拽调整日程
- **日志系统**：富文本日志编辑，AI 自动分类与标签推荐
- **看板任务**：Kanban 风格任务管理，支持拖拽流转状态
- **AI 助手**：多模型对话、日程总结（日/周/月）、智能工作流
- **本地存储**：SQLite 离线数据库，数据完全掌控在用户手中
- **快捷键**：全局快捷键支持，提升操作效率

## 五、快速开始

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装与运行

```bash
# 克隆仓库
git clone <仓库地址>
cd ai-calendar

# 安装依赖
npm install

# 开发模式启动（Web + Server）
npm run dev

# 仅启动 Electron 桌面端
npm run electron
```

### 打包为 EXE

```bash
npm run electron-build
```

输出文件位于 `release_v6/` 目录。
